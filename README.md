# uBLOG
uBLOG is a university blog web app for students and staffs. Hear from us.

## Installation
- `git clone https://github.com/dotSIS/uBLOG.git`
- `cd uBLOG`
- `pip install -r requirements.txt`
- `cd backend/uBLOG`
- `python manage.py runserver`
